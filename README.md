# 🎬 정원복 개인 포트폴리오 웹사이트 (Portfolio Website)

> **"감각적인 시선으로 이야기를 담아내고, 가치를 편집하며, 트렌드를 연출합니다."** > 프리미어 프로, 애프터 이펙트, 블렌더를 활용하여 고품질 영상을 기획·편집하는 영상 에디터 및 모션 그래픽 디자이너 **정원복**의 개인 웹 포트폴리오 사이트입니다.

---

## 📌 프로젝트 개요
본 프로젝트는 영상 에디터로서의 정체성을 시네마틱한 무드로 시각화한 **반응형 원페이지(One-Page) 포트폴리오 웹사이트**입니다. 직관적인 UI와 역동적인 인터랙션을 통해 보유 기술 스택, 제작한 영상 및 디자인 결과물, 그리고 연혁과 연락처 정보를 한눈에 파악할 수 있도록 설계되었습니다.

- **개발 버전**: V0.1.2
- **디자인 컨셉**: 글로벌 다크 무드 (시네마틱 프리미어 레드 및 무대 암전 블랙 포인트 컬러 적용)
- **타겟 브라우저**: Chrome, Safari, Edge, 모바일 브라우저 등 표준 웹 환경 완벽 호환 (반응형 지원)

---

## 🛠 사용 기술 및 개발 환경

### **Frontend & Core**
- **HTML5**: 웹 표준 준수, 시맨틱 태그 활용 및 스크린 리더 접근성 고려 (`aria-expanded` 등 반영)
- **CSS3**: 컴포넌트 기반 모듈화 설계, CSS 변수 시스템 적용 (`:root`), 애니메이션 효과 구현
- **Vanilla JavaScript (ES6+)**: 별도의 프레임워크 없이 순수 자바스크립트로 동적 기능 및 DOM 조작 구현

### **Libraries & Fonts**
- **Font Awesome (v6.0.0-beta3)**: UI 요소별 직관적인 벡터 아이콘 적용
- **Google Fonts**: 
  - `Montserrat` (영문 제목 및 헤딩용 코스믹 폰트)
  - `Noto Sans KR` (국문 가독성 중심의 본문 폰트)

## ✨ 핵심 기능 및 특징

### 1. 시네마틱 모션 타이핑 엔진 (Typing Effect)
- 인트로 섹션 진입 시, 작성자의 전문성과 핵심 가치를 나타내는 3가지 슬로건이 실시간으로 타이핑되고 지워지는 루프 애니메이션 효과가 구현되어 있습니다.
- 깜빡이는 커서(`input-cursor`) 이펙트를 더해 시각적 주목도를 높였습니다.

### 2. 포트폴리오 실시간 라이브 검색 및 필터링 시스템
- **카테고리 필터링**: 영상, 광고, 3D, 포스터, 타이틀시퀀스, 인트로, 상세페이지, 모션 등 다채로운 카테고리를 버튼 클릭 하나로 실시간 필터링할 수 있습니다.
- **키워드 라이브 검색**: 검색창 입력 즉시 작품 데이터 속성(`data-keywords`, `h3`)을 매칭하여 동적으로 카드를 필터링하며, 결과가 없을 시 전용 안내 메시지(`#no-results`)를 노출합니다.
- **상시 노출형 코스믹 버튼**: 각 포트폴리오 카드 하단에 마우스 호버 시 주황빛 네온 그로우(Neon Glow) 효과와 함께 아이콘이 전진하는 인터랙티브 버튼이 탑재되어 있습니다.

### 3. 스크롤 반응형 애니메이션 (Intersection Observer)
- 웹 최적화를 고려하여 `Scroll` 이벤트 대신 `Intersection Observer API`를 도입했습니다.
- 사용자가 스크롤을 내릴 때 포트폴리오 카드가 부드럽게 페이드인(`show`)됩니다.
- 스킬 섹션 도달 시, 각 역량 수치(`data-progress`)에 맞춰 네온 게이지 바가 실시간으로 빌딩되는 연출을 제공합니다.

### 4. 웹 접근성을 보완한 풀스크린 반응형 내비게이션
- 상단 탑바는 투명도와 블러 효과(`backdrop-filter: blur`)가 들어간 다크 무드로 디자인되었습니다.
- 모바일 및 태블릿 환경(태블릿 이하 해상도) 대응을 위한 햄버거 메뉴 및 터치 친화적 UI가 내장되어 있으며, 메뉴 개폐 시 스크린 리더가 인지할 수 있도록 `aria-expanded` 속성이 동적으로 바뀝니다.
